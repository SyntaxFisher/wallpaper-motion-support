# Wallpaper Motion Support

Wallpaper Motion is a menu bar app for animated video wallpapers on macOS.

## Download

Download Wallpaper Motion 1.0 from the [latest release](https://github.com/SyntaxFisher/wallpaper-motion-support/releases/latest).

## Features

- Play video wallpapers behind desktop icons and app windows.
- Browse and search scenic videos from Pixabay.
- Import MP4, MOV, and M4V files from your Mac.
- Keep downloaded and imported wallpapers in a local library.
- Mark favorites and export copies of videos you imported.
- Use the selected wallpaper on every connected display.
- Pause or resume playback from the menu bar.
- Match the lock screen to a still image from the selected video.
- Optionally pause playback while other windows cover the desktop.
- Launch Wallpaper Motion automatically when you sign in.

Wallpaper Motion is built in Swift for macOS.

## Get help

To report a problem or ask a question, [open a support issue](https://github.com/SyntaxFisher/wallpaper-motion-support/issues/new).

When reporting a problem, include your macOS version, Mac model, and a short description of what happened. Do not include private videos or personal information.
